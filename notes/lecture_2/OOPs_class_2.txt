//CONSTRUCTOR SYNTAX*****************

class Student {

public $name;
public $rollno;

function __construct($name, $rollno) {
	$this->name = $name;
	$this->rollno = $rollno;
 }

}
 
//creating object

	$student = new Student("Satvik", 21);
	   echo $student->name;
	   echo $student->rollno;




1. here we are using constructor which is an automated function which need not a calling, whenever we use new keyword to create an instance of a class PHP triggers it automatically.

2. while using constructor we use double underscore (__).

3. constructor is used to initialize object.
4. it flushes itself after triggered to reduce memory.
